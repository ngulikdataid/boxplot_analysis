# 📊 Boxplot Analysis - Transmitter NEC 20KW

Project ini digunakan untuk analisis data transmitter menggunakan metode visualisasi **Boxplot** dan klasifikasi status power.

---

## 🚀 Features

- 📈 Global Boxplot (All Features Comparison)
- 📊 Boxplot per Feature
- 🔍 Boxplot berdasarkan Status (LOW / NORMAL / HIGH)
- 📉 Summary Statistics
- ⚡ Power Status Classification (Engineering Threshold)
